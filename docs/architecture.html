  class BaseModelTrainer {
    <<abstract>>
    +app_config: AppConfig
    +model_cfg: SingleModelConfig
    #_pipeline: Pipeline
    #_threshold: float
    +get_model_key()* str
    +get_estimator()* estimator

    +_build_pipeline(estimator) Pipeline

    -_run_bayesian_search(pipeline, X, y)
    -_tune_threshold(pipeline, X_val, y_val)
    -_log_evaluation(...)
    -_run_name(tune) str
  }

  class LRCTrainer {
    +get_model_key() → "lrc"
    +get_estimator() → LogisticRegression
    +get_search_space() → C, penalty + SMOTE
  }
  class RFTrainer {
    +get_model_key() → "rfc"
    +get_estimator() → RandomForestClassifier
    +get_search_space() → n_estimators, max_depth
  }
  class SVCTrainer {
    +get_model_key() → "svc"
    +get_estimator() → SVC
    +get_search_space() → C, gamma + SVMSMOTE
  }
  class CatBoostTrainer {
    +get_model_key() → "cat"
    +get_estimator() → CatBoostSklearnWrapper
    +get_search_space() → depth, lr, iterations
    +_build_pipeline() → build_catboost_pipeline
  }

  class CreditRiskEnsemble {
    +pipelines: dict
    +weights: dict
    +threshold: float
    +predict_proba(X) → 1D ndarray
    +predict(X) → ndarray
    +predict_with_breakdown(X) → dict
    +optimize_threshold(X_val, y_val) → float
    +evaluate(X, y) → dict
  }

  class CreditRiskPyfunc {
    +ensemble: CreditRiskEnsemble
    +predict(context, model_input) → DataFrame
  }

  class FeatureEngineer {
    +duplicate_checking: bool
    +duplicate_amount: bool
    +fit(X, y) → self
    +transform(X) → DataFrame
  }
  class BaselineEngineer {
    +fit(X, y) → self
    +transform(X) → DataFrame
  }
  class CatBoostSklearnWrapper {
    +depth, learning_rate, iterations
    +scale_pos_weight, cat_features
    +fit(X, y) → self
    +predict(X) → ndarray
    +predict_proba(X) → 2D ndarray
  }

  class AppConfig {
    +cost_matrix: CostMatrix
    +ensemble: EnsembleConfig
    +models: dict
    +mlflow: dict
  }
  class SingleModelConfig {
    +experiment_name, registry_name
    +cv_folds, bayes_n_iter
    +use_smote, smote_type
    +column lists...
  }

Module Reference

config/core.py

Central configuration. Parses model_config.yml into validated Pydantic models. Eagerly loads config singleton at import time.

CostMatrix EnsembleConfig SingleModelConfig AppConfig load_config()

config/model_config.yml

Single source of truth. Column→encoder mappings per model, SMOTE flags, BayesSearch params, cost matrix (FP=5, FN=1), ensemble weights/threshold, MLflow URIs.

4 model configs ensemble weights cost matrix ✅ YAML typo fixed ✅ mlflow key fixed

processing/features.py

Domain-specific feature engineering for German Credit. ~10 derived features, category consolidation, binning. First pipeline step. Inherits TransformerMixin before BaseEstimator (required for sklearn ≥ 1.6 tag resolution).

FeatureEngineer BaselineEngineer ✅ MRO order fixed

processing/preprocessors.py

Pipeline factory. Assembles FeatureEngineer → ColumnTransformer → [SMOTE] → Estimator. Dedicated CatBoost path available.

build_column_transformer() build_pipeline() build_catboost_input() build_catboost_pipeline() ✅ unused import removed

processing/catboost_wrapper.py

Makes CatBoost fully sklearn-compatible for clone(), BayesSearchCV, predict_proba(). Uses super().__sklearn_tags__() for sklearn ≥ 1.8 compat.

CatBoostSklearnWrapper ✅ __sklearn_tags__ fixed

tracking/metrics.py

Evaluation functions. Asymmetric cost model (5:1 FP/FN). Returns dicts for mlflow.log_metrics().

calculate_cost() cost_scorer_fn() evaluate_model()

tracking/visualizations.py

MLflow artifact generators. Each returns a matplotlib Figure.

plot_learning_curve() plot_confusion_matrix() plot_precision_recall_curve()

training/base.py

Template Method pattern. All shared logic: MLflow setup, Bayesian search, threshold tuning, evaluation, model registration. Hook method _build_pipeline() allows CatBoost to use its own pipeline builder.

BaseModelTrainer (ABC) train() _build_pipeline() (hook) _run_bayesian_search() _tune_threshold() _log_evaluation() ✅ unused imports removed ✅ CatBoost hook added

training/train_lrc.py

Logistic Regression. SMOTE + balanced class weights. L2, liblinear.

LRCTrainer SMOTE

training/train_rf.py

Random Forest. Cost-sensitive via class_weight={0:1, 1:5}. Duplicate checking cols.

RFTrainer cost-sensitive

training/train_svc.py

SVC. SVMSMOTE oversampling. credit_amount² polynomial.

SVCTrainer SVMSMOTE

training/train_catboost.py

CatBoost. Overrides _build_pipeline() to call build_catboost_pipeline() for native categorical handling. scale_pos_weight=5.

CatBoostTrainer _build_pipeline() override native categoricals ✅ pipeline builder fixed

ensemble.py NEW

Weighted soft-voting ensemble of 4 pipelines. Cost-optimized threshold. MLflow pyfunc wrapper for deployment.

CreditRiskEnsemble CreditRiskPyfunc predict_proba() → 1D predict_with_breakdown() optimize_threshold() evaluate() ✅ evaluate() fixed (inline metrics)

predict.py NEW

Public prediction API. Loads pipelines from MLflow registry or pickle files. Single make_prediction() entry point.

load_pipelines_from_registry() load_pipelines_from_dir() make_prediction()

scripts/score_ensemble.py NEW

Standalone script to evaluate ensemble on test set, log metrics/artifacts to MLflow, register pyfunc model.

score_ensemble() ✅ __main__ guard added ✅ mlflow key fixed

scripts/export_pipelines.py NEW

Exports fitted pipelines from MLflow registry to .pkl files in app/models/. Supports version selection. Used to bridge MLflow → Streamlit/Docker.

export_pipelines() argparse CLI --output-dir, --versions

scripts/process_data.py NEW

Reads raw data/raw/german.data (space-delimited, coded attributes). Maps UCI codes (A11, A12…) to human-readable labels matching FeatureEngineer expectations. Outputs german_credit.csv.

get_mappings() create_german_credit_csv() 21 columns mapped ✅ labels match FeatureEngineer

scripts/split_data.py NEW

Stratified hash-based train/test split. Deterministic without random seeds — same row always lands in same split. Outputs train_data.csv + test_data.csv.

hash_split() validate_split() argparse CLI --test-size, --input-path

app/streamlit_app.py UPDATED

Complete Streamlit UI. Feature dicts (14 categorical + 7 numeric) with labels & help tooltips. Cached pipeline & data loaders. Two tabs: Random Samples (with true label display) and Manual Input (full form). Per-model progress bars. Cost-sensitive model explainer.
